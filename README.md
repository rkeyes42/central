# Central

The fifty-state Medicaid eligibility series — one single-file demonstration
application per state, on a deliberately constant chassis, so that everything
that differs between apps is the state itself.

**Live site:** https://rkeyes42.github.io/central/

Every member, span, and file in this series is synthetic, generated for
teaching. Program facts reflect the public record of the mid-2020s and change
continuously; verify against current CMS and state agency publications before
any reliance.
