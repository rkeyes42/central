The fifty-state Medicaid eligibility series  ·  Contact

One demonstration application per state on a deliberately constant chassis — the same synthetic population, the same register structure, the same 834 rendering — so that everything that differs between apps is the state itself. Register F of every app carries a running answer to one question: what happens in the data when Medicare arrives?

Answer numbers shown where the register F thread carries them; other states show their signatures.
Every member, span, and file in this series is synthetic, generated for teaching. Program facts reflect the public record of the mid-2020s and change continuously; verify against current CMS and state agency publications before any reliance. — Robert
