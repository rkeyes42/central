Eligibility Spans

One row per coverage-group span — the state's answer to “from when to when, and under which door did this person enter?” Click a row to open the member.

In plain EnglishA span is a promise with dates on it. It opens when a determination says yes, and it closes for a reason — a renewal that didn't come back, a nineteenth birthday, a move onto Medicare. An open end date means the promise is still standing. Maryland's spans carry the coverage group, because the group decides everything downstream: whether HealthChoice enrollment is required, what the benefit package looks like, and which rules end it.

HealthChoice Enrollment

One row per delivery-system span: which MCO held the member, or whether the member sat in fee-for-service or the REM carve-out. Click a row to open the member.

In plain EnglishA delivery span is not just a plan name — the plan is the value; the span is that value held over a date range, with a reason it began and ended. The value is one of three kinds: an MCO (a capitated plan), FFS (no plan — the state paying claims directly, a positive fact with its own dates, not a gap), or REM (FFS plus a dedicated case manager). And eligibility and delivery run on two different clocks: eligibility can be granted backward (retroactive months are always fee-for-service, because you cannot capitate the past), but MCO enrollment only runs forward, from the first of a month. That is why a member's history can honestly read MCO → FFS → MCO — and why members leaving HealthChoice for Medicare or REM show an MCO span that ends while the eligibility span sails on.

What the File Says

Maryland really does speak X12N 834 — the state's enrollment file to its HealthChoice MCOs. Pick a member and read the transaction the MCO would receive for that member's most recent enrollment event, segment by segment, with a plain-language note on every line.

In plain EnglishAn 834 is the state telling a health plan: this person is yours now (or isn't anymore). It carries who the person is, where they live, when coverage starts or stops, and — in Maryland's file — the coverage group that explains why. Fee-for-service members generate no MCO 834 at all: for them, the file below is the enrollment record the state keeps for itself, shown in the same shape for teaching.
Members are grouped by register; the transaction is rebuilt live from the member's actual spans.
